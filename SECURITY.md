# Security Policy

## Reporting Security Issues
**Do not** report security issues publicly.

Use GitHub's private vulnerability reporting or contact maintainers privately.

## Supported Versions
- v2.2.0: ✅ Supported
- v2.1.x: ⚠️ Critical issues only
- Earlier: ❌ Not supported

## Security Practices
- Input validation
- No hardcoded secrets
- Regular audits
